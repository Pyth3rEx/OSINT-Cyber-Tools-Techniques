Custom files installed from this repo:

- Firefox bookmarks HTML: /usr/local/share/osint/browser/bookmarks.html
- User copy of bookmarks: ~/Documents/OSINT/Firefox/bookmarks.html
- Shell profile: ~/.zshrc
- Helper scripts:
  - /usr/local/bin/ip_panel.sh
  - /usr/local/bin/WireGuard_cfg_batch_import.sh

To import the bookmarks into Firefox:

1. Open Firefox.
2. Open the bookmarks manager.
3. Use "Import and Backup" -> "Import Bookmarks from HTML".
4. Select ~/Documents/OSINT/Firefox/bookmarks.html.
